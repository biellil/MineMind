## Formato da coordenada

| Option | Description | Selected |
|--------|-------------|----------|
| Híbrido: abs + dist + Δaltura | `oak_log×4 próx(12,70,-5) 8m Δy+7` — mantém x,y,z p/ navigate e dá altura legível | ✓ |
| Absoluta (x,y,z) + distância | Consistente com pos/navigate, sem altura relativa explícita | |
| Relativa (Δx,Δy,Δz) | Direção/altura imediatas, mas LLM reconverte p/ navigate | |

**User's choice:** Híbrido (abs + dist + Δaltura)
**Notes:** Valor real da posição é o LLM raciocinar alcançável; Δaltura entrega isso direto e ainda preserva x,y,z para navegar a um ponto.
