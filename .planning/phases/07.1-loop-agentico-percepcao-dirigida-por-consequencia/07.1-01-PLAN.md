Criar src/cognition/trigger-bus.ts com o seguinte contrato EXATO:

**Tipo TriggerEvent** (union string-literal):
```typescript
export type TriggerEvent = 'actionFinished' | 'nightFell' | 'dayBroke' | 'hostileNearby' | 'stuck' | 'hungry'
```

**Interface TriggerConfig**:
```typescript
export interface TriggerConfig {
  hostileRadius: number       // distância em blocos para detectar mob hostil (default: config.hostileRadius)
  hostileDebounceMs: number   // debounce para hostileNearby (default: config.hostileDebounceMs)
  hungryThreshold: number     // food <= this → emite hungry (default: config.hungryThreshold)
}
```

**Classe TriggerBus extends EventEmitter**:
- Importar `EventEmitter` de `'node:events'` (zero dependências novas)
- Campos privados de edge-state: `_wasDay: boolean = true`, `_lastFood: number = 20`, `_hostileDebounce: ReturnType<typeof setTimeout> | undefined`
- Método público `setupMineflayerListeners(bot: Bot, cfg: TriggerConfig): () => void`

**Implementação dos edge-detectors dentro de setupMineflayerListeners:**

1. **hungry** — assina `bot.on('health', onHealth)` (não 'physicsTick', não polling):
   ```typescript
   const onHealth = () => {
     if (bot.food <= cfg.hungryThreshold && this._lastFood > cfg.hungryThreshold) {
       this.emit('hungry', { food: bot.food })
     }
     this._lastFood = bot.food
   }
   ```
   Histerese embutida: só emite quando cruza o limiar para baixo (não a cada tick abaixo).

2. **nightFell / dayBroke** — assina `bot.on('time', onTime)` com flag:
   ```typescript
   const onTime = () => {
     const isDay = bot.time.timeOfDay < 13000
     if (!isDay && this._wasDay) this.emit('nightFell')
     if (isDay && !this._wasDay) this.emit('dayBroke')
     this._wasDay = isDay
   }
   ```
   Uma comparação booleana por tick (custo mínimo). NUNCA emitir sem cruzar a borda.

3. **hostileNearby** — assina AMBOS `entitySpawn` e `entityMoved`:
   ```typescript
   const checkHostile = () => {
     const botPos = bot.entity?.position
     if (!botPos) return
     const hasHostile = Object.values(bot.entities).some(
       (e) => (e as any).kind === 'Hostile mobs' &&
              e.position.distanceTo(botPos) < cfg.hostileRadius
     )
     if (hasHostile) {
       clearTimeout(this._hostileDebounce)
       this._hostileDebounce = setTimeout(
         () => this.emit('hostileNearby'),
         cfg.hostileDebounceMs
       )
     }
   }
   bot.on('entitySpawn', checkHostile)
   bot.on('entityMoved', checkHostile)
   ```
   NUNCA assinar `bot.on('entityMoved', ...)` sem debounce (D-13 — anti-tick-disfarçado).

4. **actionFinished** e **stuck** — NÃO são derivados de eventos nativos do bot. São emitidos externamente (pelo nó execute no Plan 3). O TriggerBus apenas fornece o canal (é um EventEmitter). NÃO criar handlers para eles aqui.

**Retorno de cleanup obrigatório**:
```typescript
return () => {
  bot.off('health', onHealth)
  bot.off('time', onTime)
  bot.off('entitySpawn', checkHostile)
  bot.off('entityMoved', checkHostile)
  clearTimeout(this._hostileDebounce)
}
```
