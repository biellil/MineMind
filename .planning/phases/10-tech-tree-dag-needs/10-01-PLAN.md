**Lógica interna (seguir esta ordem — D-01, Pitfall 1):**
1. `if (memo.has(targetItem)) return memo.get(targetItem)!`
2. `if (depth >= 8) return { unresolvable: true }` (D-04)
3. Verificar SMELT_MAP[targetItem] PRIMEIRO (Pitfall 1 — evitar ciclo iron_ingot):
   - Se presente: `sourceItem = SMELT_MAP[targetItem]`
   - Sub-goals: `resolveDag(sourceItem, bot, memo, depth+1)` + `smelt:sourceItem` goal
   - Se sourceResult for `{ unresolvable: true }`, propagar
   - Sub-goal smelt: `{ id: \`smelt:${sourceItem}\`, kind: 'smelt', dependsOn: [\`gather:${sourceItem}\`], ... }`
4. Tentar `bot.recipesAll(itemId, null, /* craftingTable= */ true)` (Pitfall 4 — usar recipesAll, não recipesFor):
   - `itemId = bot.registry.itemsByName[targetItem]?.id`
   - Se `itemId === undefined`, ir para fallback gather
   - Recipes vazias → fallback gather
   - Escolher UMA receita: filtrar receitas cujos ingredientes estejam na gatheringLadder (config.gatheringLadder) OU usar a primeira disponível
5. Para a receita escolhida:
   - Extrair ingredientes via `recipe.delta.filter(d => d.count < 0).map(d => bot.registry.items[d.id]?.name).filter(Boolean)` (Code Example 2)
   - Deduplicate ingredientes
   - Se `recipe.requiresTable`: adicionar sub-goal `{ id: 'ensure:crafting_table', kind: 'ensure', dependsOn: [] }` (Pitfall 2)
   - Resolver recursivamente cada ingrediente: `resolveDag(ingredient, bot, memo, depth+1)`
   - Se algum ingrediente retornar `{ unresolvable: true }`, tentar próxima receita alternativa; se todas falharem, propagar unresolvable
   - Montar lista topológica: [...subGoalsDeIngredientes, ...ensureGoals, goalDoItem]
   - O goal do item: `{ id: \`craft:${targetItem}\`, kind: 'craft', priority, dependsOn: [últimoSubgoalDePré-req], source: 'need', committedAt: now }`
6. Fallback gather (sem receita e não está em smelt map):
   - Retornar `[{ id: \`gather:${targetItem}\`, kind: 'gather', dependsOn: [], ... }]`
7. Salvar no memo: `memo.set(targetItem, result)`

**Priority dos sub-goals:** herdar a priority do goal pai (passada como parâmetro `basePriority: number = 0.8`). Adicionar ao parâmetro da função.

**Timestamp:** receber `now: number = Date.now()` como parâmetro (módulo puro — sem Date.now() direto).

```typescript
export function selectGoal(
  current: Goal | null,
  candidates: Goal[],
  ctx: SelectGoalContext,
  cfg: MotivationConfig,
  completedIds: Set<string> = new Set(), // D-06: filtro de dependsOn
): Goal | null {
  // D-06: filtrar ANTES da histerese — goals bloqueados por deps não satisfeitas são removidos
  const unblocked = candidates.filter(g =>
    g.dependsOn.every(depId => completedIds.has(depId))
  )

  const preempt =
    ctx.survivalCritical || (ctx.disposition === 'ASSISTANT' && ctx.playerRequestPending)

  if (current && !preempt) {
    const best = bestCandidate(unblocked)
    if (!best || best.priority < current.priority + cfg.hysteresisMargin) return current
  }
  return bestCandidate(unblocked) ?? current
}
```
